# Large-Scale Analysis of US Traffic Accidents (2016–2023)

## Overview
This project was developed for the *Distributed Data Analysis and Mining* course as part of my Master's degree in Data Science and Business Informatics at the University of Pisa.

The project focuses on the analysis of a large-scale US traffic accidents dataset (2016–2023) using distributed computing techniques.

It covers the main steps of a data mining pipeline, including data preparation, clustering, classification, regression, and explainable AI, with the goal of exploring patterns and supporting predictive analysis on accident data.
